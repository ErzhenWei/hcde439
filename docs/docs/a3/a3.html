Resistance Calculation

The calculations for the normal resistance in the circuits show that a minimum of 150 Ohm is needed to keep LEDs safe; for clarity, I used 220 Ohm resistance. And for the R2 resistance, I used geometric mean to calculate a minimum of 150k Ohm is needed; for clarity, I used two 100k Ohm resistance in series, and got a total of 200k Ohm.

Threshold Values

By reading the serial monitor's values, I found that the reading at voltage divider is about 1010 when it is bright, and about 930 when it is dark. Combining these two values, I decided the voltage divider reading threhold value should be 950. And since the readings under both conditions are pretty high, I used the PWM signals directly as the LED brightness value.

Firmware


// Define pin hole number for two LEDs
// LED that turns on when dark
const int darkLedPin = 9;
// LED that turns on when bright
const int brightLedPin = 11;

// Define analog input pin
const int sensorPin = A0;

// Variable to store analog sensor reading
int sensorValue = 0;

// Variable to store mapped brightness
int ledBrightness = 0;

void setup(){
  // Set the LED pins as output
  pinMode(darkLedPin, OUTPUT);
  pinMode(brightLedPin, OUTPUT);
  // Start serial communication
  Serial.begin(9600);
}

void loop(){
  // Read voltage at the divider (detect the illumination)
  sensorValue = analogRead(sensorPin);

  // Print the sensor valuel
  Serial.print("Light sensor value:");
  Serial.println(sensorValue);

  // Convert 10-bit sensor reading to 8-bit PWM output
  ledBrightness = map(sensorValue, 0, 1023, 0, 255);

  // dark condition (light level is low)
  if (sensorValue < 950){
    // Increase the brightness of dark LED as it is dark
    analogWrite(darkLedPin, ledBrightness);
    // Turn off the bright LED
    analogWrite(brightLedPin, 0);
  // bright condition
  }
    else {
          // Increase the brightness of bright LED as it is dark
        analogWrite(brightLedPin, ledBrightness);
        // Turn off the dark LED
        analogWrite(darkLedPin, 0);
  }

  // delay 100 ms
  delay(100);
}
